# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/).

Per-addon changelogs (where they exist) live in each addon's own folder (e.g. `ElvUI_ProfileConverter/CHANGELOG.md`, `SpellActivationOverlay/changelog.md`). This root changelog tracks repo-level and cross-addon changes.

## [Unreleased]

- Back-ported SpellActivationOverlay v2.7.2 from Cataclysm+ to WoW 3.3.5a
